Objetivo:  
Conocer el funcionamiento de un validador en Angular para asegurar la integridad de los datos dentro de un formulario reactivo antes de que sean procesados o enviados al servidor.  

Objetivo de la entrega: Demostrar la capacidad de asegurar la integridad de los datos enun formulario reactivo de Angular , garantizando una comunicación eficiente con un backend en Spring Boot sin comprometer la experiencia de usuario (UX).
